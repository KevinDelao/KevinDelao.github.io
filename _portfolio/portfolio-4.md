A fun project I wanted to work on to show the history of my favorite places around LA. Additionally, I wanted to work on this project to gain experience working with multiple API's.
By working on this project I was able to gain skills in learning to read documenation for API's instead of just following tutorials. After reading through documenation I was able to learn
how work with both the Google Maps API and the Wikipedia API and have them communicate with each other. The end result was a project that marks the location of 10 of my favorite places
around LA and when a mouse cursor hovers over any of the markers, a window below the map will show up and display info from Wikipedia about that location.

In the future I plan to develop this project to automatically select the 10 most popular places based on the location of a given user in order to not just show places around LA.
