After completing both my REST API project and Los Angeles Favorite Places project, I came to realize that I did not do much testing for these projects. From my past research experience, I have become
aware of how important testing is to any algorithm developed. Because of this, I am now going to go through all of my past projects to implement thorough testing for all of my code. My goal will be to gain more experience 
in using testing libraries commonly used in industry and to ensure that anyone who uses my code will have the assurance that the code I developed is extensively tested for reliability.
